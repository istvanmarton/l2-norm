The L2_eta_70 dictionary contains:

W70i.txt: The 70x70 dimensional integer witness matrix 
grassc_3_1_70.txt: The coords of the best known chordal packing in 3 dimensions of 70 1-dimensional lines
http://neilsloane.com/grass/dim3/grassc.3.1.70.txt

output_L2.txt: This is the output of the Haskell code Ln for order 2 by Peter Divianszky. The output is the L2 norm of the matrix W70i.
The code is available at https://github.com/divipp/l2-norm

eta_70.m: Matlab code for the computation of the Q and S values and the resulting eta critial value, 
which is given by eta_crit = (L2-S)/(Q-S). 

output_eta: Result file of the Matlab code eta_70.m. The computed eta critical value is given by eta_crit = 0.6376.
